База данных проекта находится на бесплатном хостинге db4free.net. Поскольку хостинг бесплатный существуют ограничение на добавление данных (размер не должен превышать 200 мб). База данный может быть удалена владельцами хостинга в результате разного рода сбоев, что влечет за собой некорректную работу программы. 

Бэкап базы данных находится в папке DataBase backup.

Если не удается войти с телефона следует поменять IP в админ панели программы на ПК. Также при установке пакета .apk будет создан ключ. Если ключ на аккаунте пустой, он будет применен. Если ключ уже есть в приложение войти не получится

При изменении или удалении данных учитывается ID. В случае если ID не ввести программа выдаст ошибку.
